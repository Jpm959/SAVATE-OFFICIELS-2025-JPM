OPTION 1 : Via le bouton "Deploy" (RECOMMANDÉ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Cliquez sur le bouton "🚀 Deploy" en haut à droite
2. Confirmez le déploiement
3. Attendez 1-2 minutes que le déploiement se termine
4. Vous recevrez une URL de production, par exemple :
   https://votre-projet-prod.emergent.app

5. Ajoutez le nom du fichier à la fin de cette URL :

   Pour l'application :
   https://votre-projet-prod.emergent.app/savate-officials-FINAL.html

   Pour l'archive ZIP :
   https://votre-projet-prod.emergent.app/SAVATE-OFFICIALS-LIVRAISON-FINALE.zip

   Pour la page de téléchargement :
   https://votre-projet-prod.emergent.app/PAGE-TELECHARGEMENT-DIRECTE.html

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Vous pourrez alors le copier-coller et le décoder sur votre ordinateur.
